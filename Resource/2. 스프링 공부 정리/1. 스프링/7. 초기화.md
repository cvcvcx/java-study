
> 기본적으로 스프링빈은 즉시초기화한다. 하지만 지연해서 초기화 하는 방법도 존재한다

# 📚지연초기화와 즉시초기화의 비교

| 구분       | 지연초기화                      | 즉시초기화                  |
| ---------- | ------------------------------- | --------------------------- |
| 초기화시점 | 빈이 사용될 때                  | 프로그램이 실행될 때        |
| Default    | 기본설정아님                    | 기본설정임                  |
| 에러발생시 | 런타임에러(프로그램실행중 에러) | 프로그램 실행할 때 에러발생 |
| 메모리사용 | 적음                            | 항상 모든 빈이 생성됨       |                         |

웬만하면 즉시초기화를 사용하자!
